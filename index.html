Portfolio
C++ BM3D Optimization
C++

BM3D Denoising Algorithm Optimization for Intel CPU

Final year project in collaboration with ViTrox Technologies. This project involved optimizing the computational performance of the BM3D image denoising algorithm, which is known for its excellent noise suppression and detail preservation but suffers from high processing costs. The optimization focused on improving execution efficiency for CPU-based systems through SIMD-based vectorization, memory alignment, and loop unrolling techniques. The implementation reduced instruction counts from 754 million to 32 million and improved processing time by up to 95%, while maintaining original PSNR values. The project also explored multi-threading and memory pre-allocation strategies to enhance CPU core utilization and reduce bottlenecks. Testing covered various image sizes and CPU profiling metrics.
